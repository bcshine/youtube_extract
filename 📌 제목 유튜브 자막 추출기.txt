📌 제목: 유튜브 자막 추출기

🧩 목표 기능
- 유튜브 영상 URL을 입력하면 자막을 추출해서 보여줍니다.
- 자막은 textarea에 출력됩니다.
- 자막 내용을 복사할 수 있는 '복사하기' 버튼이 있습니다.
- 모바일 반응형 디자인으로 만듭니다. 

🧱 구성 요소
1. 입력창: 유튜브 URL을 입력하는 input
2. 버튼 1: '자막 추출' 버튼을 누르면 자막을 가져와 textarea에 출력
3. textarea: 자막을 출력하는 영역 (읽기 전용)
4. 버튼 2: '복사하기' 버튼을 누르면 textarea 내용을 클립보드에 복사

🎨 스타일
- 전체 레이아웃은 중앙 정렬, 너비는 600px 이하로 제한
- 배경은 밝은 회색(#f4f4f4), 박스는 흰색 배경과 그림자 처리
- 버튼은 파란색(#007BFF) 배경, 흰색 글씨, hover 시 색상 변경

⚙️ 동작 방식 (JavaScript)
- 입력된 유튜브 URL에서 video ID를 추출
  예: https://www.youtube.com/watch?v=abc123 → abc123 추출
- video ID를 기반으로 가상의 자막 API를 호출 (임시 JSON 사용 가능)
- 성공 시 자막 데이터를 textarea에 출력
- 복사하기 버튼을 누르면 textarea의 내용을 클립보드에 복사

💡 참고:
- 실제 유튜브 API는 보안 문제로 클라이언트에서 직접 접근이 어려우므로,
  예시 자막을 JSON 형태로 미리 작성하거나 테스트용 API를 활용하세요.
